All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-07

### Added
- Initial release of Rent Curves landing page
- Two landing page versions:
  - `index.html` - Casino-style with emerald/gold theme
  - `index-3d.html` - Neo-futuristic with 3D animations
- Interactive crop wheel with 6 crops (Corn, Soybeans, Wheat, Canola, Cotton, Rice)
- Auto-rotating wheel with spotlight effect
- Hover interactions (slow rotation, scale up)
- Click-to-select with particle burst animation
- Re-selection support for switching crops
- 5 input value cards (Price, Yield, Cost, High, Avg)
- Dramatic rent calculation reveal with gradient text
- Anime.js number counting animations
- Three.js particle background
- GSAP stagger and elastic animations
- Fully responsive design
- Mobile hamburger menu
- Glassmorphism UI components
- Casino-style engagement without being gaudy
- Client specification documentation

### Technical Stack
- GSAP 3.12.5 for animations
- Three.js r128 for 3D background
- Anime.js 3.2.2 for micro-interactions
- Google Fonts (Space Grotesk, Syne)
- Pure CSS with custom properties
- No build step required (CDN dependencies)
